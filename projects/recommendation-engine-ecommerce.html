Recommendation Engine for E-Commerce

A personalised recommendation engine for an e-commerce platform combining classical association rule mining with deep learning to surface relevant products to each user. Led a six-member team as core developer and integrator, delivering a production-ready system on time.

Project context

Why this project mattered

Generic product listings do not convert. The business need was a recommendation layer that could surface products a given customer was genuinely likely to buy — based on purchase history, browsing behaviour, and patterns across similar users. Without personalisation, product discovery relied entirely on manual browsing, and conversion rates reflected it.

I led the engineering team as core developer and integrator across a six-member team, responsible for the recommendation logic design, the deep learning integration, and the overall code quality and delivery coordination under agile methodology.

How I achieved it

Execution approach

01 Association rule mining

Designed the core recommendation logic using the Apriori algorithm in TensorFlow — mining frequent itemsets from purchase history to surface co-bought and co-browsed product associations across the catalogue.

02 Deep learning integration

Combined the Apriori-based association approach with deep learning models built in Keras and TensorFlow, enabling the system to incorporate user-level behavioural patterns and sequence information that classical methods could not capture alone.

03 Accuracy improvement

Applied transfer learning techniques on top of the baseline model to improve recommendation relevance — fine-tuning on domain-specific purchase and session data to align outputs more closely with real user intent.

04 Production delivery

Applied RPA techniques and agile practices to streamline the development and deployment workflow, reducing integration friction and enabling faster iteration between engineering and business validation cycles.

Architecture

System design

Data layer

Purchase history, browsing sessions, and product catalogue data ingested and preprocessed into user-item interaction matrices and transaction itemsets.

Association mining

Apriori algorithm extracts frequent itemsets and generates confidence-ranked association rules — forming the baseline co-purchase recommendation signal.

Deep learning layer

Keras and TensorFlow models learn user-level embeddings and sequential purchase patterns, augmenting the rule-based signal with personalised affinity scores.

Scoring and ranking

Combined scores from association rules and deep learning predictions are ranked per user and filtered by inventory availability before serving.

Serving layer

Ranked recommendations served via API to the e-commerce frontend, supporting both real-time personalised product grids and triggered recommendation emails.
